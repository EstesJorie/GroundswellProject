# Python Dependencies 
- OS
- Groq
- Python-Dotenv
- Flask 
- Flask Cors

# Dependency Install

## Conda
- conda install conda-forge::groq
- conda install conda-forge::python-dotenv
- conda install anaconda::flask
- conda install anaconda::flask-cors


## PIP 
- pip install groq
- pip install python-dotenv
- pip install flask 
- pip install flask-cors
